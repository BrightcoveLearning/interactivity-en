Branching Video Overview

Branching or Choose-Your-Own-Adventure is a popular type of interactive video. In a nutshell, it brings a viewer to different parts of a video, or new video entirely, based on how they interact with it. You can read more on the basics of Branching here.

If you created a Lesson via HapYak's LTI integration you can do branching in two ways.

In-Video Branching

You do In-Video branching just as you would in a single project in the HapYak portal. Detail here.

Video to Video Branching

You can branch from video to video by following this technique.

  1. Create a lesson.
  2. Create a second lesson (the next video where you'd like the viewer to branch to).
  3. View the second lesson as a student. Copy the URL of that lesson.
  4. Go back to the first lesson. Add an annotation (text, hotspot or image) that the viewer would click to branch to the next video. In Link (ON CLICK EVENT) / Load external URL:, paste the URL to the second lesson. Click Submit. Click Done.
  5. Test by viewing this first lesson as a student. Clicking on the annotation should load the second lesson.
Note: You can use this same technique in a Quiz annotation by adding the second lesson URL to the Pass/Fail actions field.
