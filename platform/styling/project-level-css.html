Note: This topic applies to the legacy version of Brightcove Interactivity (HapYak). Click here for the new documentation.
Note: All CSS URLs need to be formatted with no uppercase letters to ensure proper functionality.

To apply custom CSS to one project:

  1. Upload the CSS file to your server. Copy the link to it.
  2. Edit the project you are working on.
  3. Click SETTINGS in the top right corner of the page.
  4. Locate the Custom Annotation CSS (URL) field and add the URL to your CSS file (ensure the protocol, http: or https:, is removed from the URL).
  5. Click SUBMIT.
