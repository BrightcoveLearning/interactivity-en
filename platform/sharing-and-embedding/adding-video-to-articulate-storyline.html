Note: This topic applies to the legacy version of Brightcove Interactivity (HapYak). Click here for the new documentation.
  1. In the Brightcove Interactivity project, generate a landing page. Copy the URL of the landing page.
  2. In Articulate, Select Articulate --> Web Object to launch the Insert Web Object dialog.
  3. Paste the URL of the landing page.
  4. Select Display in Slide.
