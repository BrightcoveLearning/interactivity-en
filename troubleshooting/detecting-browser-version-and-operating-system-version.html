Occasionally, Brightcove Support will ask you to send the browser version and operating system version on which you are seeing an issue.  This helps the team open up the project in the same environment you are using, replicate the issue and detect the root cause of it.

To find out which browser version you are using simply open whatsmybrowser.org in the browser.
