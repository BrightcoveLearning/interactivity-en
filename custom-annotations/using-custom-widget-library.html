The Custom Widget Library is a set of more advanced interactions that you can add with one click to your Brightcove Interactivity projects. There is no need to write any HTML, JavaScript or CSS code.

Note that all custom annotations are in Beta and should be tested before going live.

 

Workflow

  1. While editing a project, click the Add Iframe Icon. 
  2. Click the custom widget you'd like to add. Set start time, end time, Pause, Gate, and any other annotation options as normal.
  3. Follow the widget specific instructions detailed below.
