In the HapYak portal, you can use HapYak's Bulk Annotation Edit functionality to update, copy, or delete multiple annotations simultaneously. 

To use the Bulk Annotation Edit navigate to a project page and scroll down to the list of included annotations. Hold down the Command button (on Mac) or CTRL button (on Windows) and click the desired annotations. Each selected annotation will highlight when clicked. 

After selecting all the annotations, release the Command/CTRL button and you will be taken to the Multi-Edit Modal, where you can update the start and end time, copy, or delete the selected annotations.

After making your changes, simply click the Submit button to apply and save.
