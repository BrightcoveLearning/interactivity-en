Sharing a project generates embed code or a URL so it can be seen by viewers. If you are publishing a Brightcove video, publishing and sharing should be done using a HapYak-enabled Brightcove Player using the Media module in Video Cloud Studio.

If you are unable to publish using Video Cloud, follow these steps to publish from within the Brightcove Interactivity Editor.

  1. Login to Brightcove Interactivity Studio.
  2. In the left navigation, click Manage.
  3. Click on a project or experience to open it.
  4. Click SHARE.
  5. Select a Share Type. The following types are available:
    • HapYak Wrapper
    • Landing Page
    • Embed.ly Link
    • Brightcove Player
    • SCORM

Brightcove Player

When publishing to a Brightcove Player, it is recommended that you publish using the Media module in Video Cloud Studio.

  1. Make sure you have created a HapYak-enabled Brightcove Player.
  2. Login to Video Cloud Studio.
  3. Open the Media module.
  4. Select a video.
  5. Click Publish and embed... Ë‡ > Publish to web.

Make sure you select a HapYak-enabled Brightcove Player. For more information, see Publishing Videos and Playlists with the Media Module.

HapYak wrapper

Sharing using a HapYak Wrapper is useful when you need to embed a project or experience on a website, CMS, or LMS. This option will generate the iframe embed code that can be embedded in other places.

To generate a landing page, follow these steps.

  1. For the Share Type, click HapYak Wrapper.
  2. Select a Size.
    • Responsive - Recommended. This option ensures that the embed code will adapt to the height and width of the the parent element in which the video is embedded. This will provide the optimal viewing experience across all devices.
    • Fixed Size - Enter specific player dimensions. The lock icon locks the aspect ratio when you manually adjust the dimensions. A Fixed Size embed will always remain the same size and may only be partially viewable on some devices or in smaller browser windows.
  3. Copy the embed code to use in your application. You can also click PREVIEW to open a preview of the project in another browser tab.

Landing page

Sharing to a landing page will generate a dedicated, static URL that any viewer can access. This is a quick and easy way to share your project with others, for example, to get feedback from collaborators or clients or more broadly in an email, on Slack, or any other channel.

To generate a landing page, follow these steps.

  1. For the Share Type, click Landing Page.
  2. Click GENERATE to generate a landing page.

If you would like to customize the landing page by adding a header, footer or social share icons for example, click EDIT. This will open the Experience Editor for that video. Click SETTINGS and customize the social sharing and branding settings as needed.

Embed.ly link

Embed.ly is a service that lets hundreds of systems and platforms (like Jive, Medium, reddit, Disqus, AOL, Digitas, Acquia, Trello, Udemy, etc.) make it easier for people to embed content (like interactive videos).

Brightcove Interactivity is an Embedly provider. What this means is if your system supports Embedly, then all you have to do is copy and paste the Embedly link and it will figure out the rest and embed your Brightcove Interactivity project for you.
