To apply the Custom CSS to one Project:

1. Upload the CSS file to your server. Copy the link to it.

2. Go to the project you are working on.

3. Click Settings in the top right corner of the screenï»¿

4. In the text box labeled, Custom Annotation CSS URL add your css url (ensure the protocol, http: or https: is removed from the url).

5. Click Submit
