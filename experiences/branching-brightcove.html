Before authoring any annotations do the following:

  • Ensure the HapYak Plugin for Brightcove is setup on a Brightcove player.
  • Ensure that a project has been auto-created in HapYak for each Brightcove video you'd like to use in the branching experience.

Once you have done that you simply use the Branching Experience editor to author.

If you have used the Brightcove Playlists method to author branching experiences in the past, that is now replaced by this simplified, more streamlined authoring method.
