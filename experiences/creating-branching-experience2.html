The Branching Experience allows you to quickly string together multiple videos and add annotations that let the viewer jump from video to video. This creates a "choose your own adventure" type experience for your viewers.

The Branching Experience editor dramatically streamlines the editing process, cutting work time from hours to minutes and is available out-of-the-box to all plans.

Note: If you have used the Brightcove Playlists method to author branching experiences in the past, that is now replaced by this simplified, more streamlined authoring method.

Creating a new experience

Follow these steps to create a new Branching Experience:

  1. Login to Brightcove Interactivity Studio.
  2. In the left navigation, click Manage.
  3. At the top of the page click Experiences.
  4. Click New Experience.
  5. Enter a name for the experience.
  6. In the list of experience types, click Slide Sync.
  7. Choose a layout.
  8. Click NEXT.
  9. Enter a Title for the video.
  10. Choose a video, there are 4 options:
    • Sample Video - Use a sample video
    • URL or Embed Code - Enter the embed code or URL for a video
    • Upload - Drag and drop or select a new video (must be an .mp4 file)
    • Choose existing - Choose an existing project
  11. Click NEXT. The New Experience confirmation dialog will display.
  12. Click CREATE. The Interactivity Editor will open with the selected video(s) loaded.
  13. Add PDF slides to the experience by dragging a file into the drop zone or by clicking anywhere in drop zone and browsing to select a file.
    • The slides must be in PDF format
    • The first page in the PDF will appear and sync to the 0 second point in the video

Create

  1. Click the Experiences Tab
  2. If needed, click "New Experience".
  3. Name the experience. Click "Branching" Type. Click "Next".
  4. Add Projects. You can choose existing projects or create new ones. (As a reminder a Project in Brightcove Interactivity is one interactive video). Additional projects can also be added later.
 

Edit

  1. You will now add all the annotations you need, You'll most likely be adding text or hotspot annotations that branch from video to video. To do so, add the annotation, click "link", then Jump to video: and select the video from the list.
  2. If you select "Jump to Video" you can then select "Jump to video time" to go to specific point in the destination video. 
  3. If you just select Jump to video time without a Jump to Video selected it will simply jump to that time in the current video.
  4. At the bottom you'll see the full list of videos. Simply drag and drop to re-arrange the order of videos or click Edit to edit a different video.

Share

Landing Page -  On the Experience page, click "View". This will open up a dedicated landing page URL that you can share with anyone. This URL is constant no matter how you re-edit or change settings on the experience, so share away!

Embed - On the Experience page, click "Share" then Share Type. Copy the embed code and paste into your CMS, LMS, website, etc.... 

Settings

  • Auto Progress - Automatically progresses to the next video in the playlist when the previous video ends.
  • Auto Play - Automatically plays each video in the playlist when a viewer clicks to it or when they progress to it automatically.
  • Custom CSS (URL) - Apply custom styling to the entire experience. Read more here.
  • If you would like to hide the project titles from the viewer add this to the experience's custom css field: //hapyak_demos.s3.amazonaws.com/css/hideExperienceTitles.css
  • Social Sharing - Standard social sharing bar at the top. Select the social channels you'd like to include. Add an optional message.
  • Branding - Add a header or footer to the landing page. Include HTML code directly here.
