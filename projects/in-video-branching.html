Brightcove Interactivity offers two techniques for creating branching or "choose your own adventure" (CYOA) content.

  • Video to video branching - This technique uses a Branching Experience where multiple videos are strung together. The viewer jumps from video to video using annotations.
  • In-Video branching - This technique uses a single video. The viewer jumps from poin to point in the same video using annotations.

Both work equally well, it's a personal preference based on your production process and how your videos are already produced.

In-Video branching sample

The following is an example of in-video braching. The main menu "page" will loop until an image is clicked. Clicking an image will jump to that part of the video. When that part of the video is complete, a triggered action will send the video back to the main menu page. The "Main Menu" text is transparent overlay that displays throughout the video and will jump to the main menu when clicked.

Branching withing a single project

Follow these steps to add branching to a project:

  1. Login to Brightcove Interactivity Studio.
  2. In the left navigation, click Manage.
  3. Click on a project to open it.
  4. Add a chapter marker at each point in the video where a choice begins. The chapter markers will be invisible to viewers.
  5. For each "choice point", add a text, image or transparent overlay annotation that the viewer will click on to jump to different parts of the videos. Make sure the start time and duration for each of these annotations is identical.
  6. On one of the choice annotations, set Action to take if the viewer doesn't click to Loop back.
  7. On each annotation set Link / Actions to Jump to video time and select the correct Chapter or enter a custom time.
