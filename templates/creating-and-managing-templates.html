A template is a track (or group) of annotations that can be applied to multiple projects. Templates can be used to quickly apply identical annotations to multiple videos with one click.

Multiple template tracks can be added to a project. If the template track is updated, Brightcove Interactivity will automatically update all projects using the template. Templates are useful for any features that should be added to a large number of videos. For example, adding a logo to all of the company's videos, adding a "skip-ahead" button to a series of videos that have the same pattern, or adding a call to action to all videos with a similar subject.

Creating a new template

  1. Login to Brightcove Interactivity Studio.
  2. In the primary navigation, click Create and then NEW TEMPLATE or click Manage, click the Templates tab and then click New Template.
  3. Enter a Title for the template.
  4. Click NEXT.
  5. Brightcove Interactivity automatically loads a sample video source. You can change this video by clicking Configure and by entering a new URL in the Video Source field.
  6. Add the appropriate annotations.
    Tip: If you set the start time to -20, the annotation will appear 20 seconds before the end of the video. This means it will dynamically adjust for every project the template is applied to. Likewise, 25% will set the start time to 25% of the way into the video, no matter the length.

Applying a template to a project

To apply a template to a project, follow these steps:

  1. Click on a project title to open it.
  2. Click SETTINGS.
  3. In the PROJECT section, locate the Templates Applied setting.
  4. Click the edit icon to open the Configure project templates dialog.
  5. Click on one or more templates to apply them to the project. Selected templates will appear at the top of the dialog.
  6. Click SUBMIT.

Copying templates

On the list of templates, hover over the right side of the template and click the copy icon ().

This will create an exact clone of the template including all annotations, tags, and custom CSS, titled "Copy of {template_name}".

This is a great way to save time when creating new templates that are similar to ones already created. Simply adjust the annotations as needed.

Deleting templates

To delete a template, find the template on the template list, click the delete icon (
