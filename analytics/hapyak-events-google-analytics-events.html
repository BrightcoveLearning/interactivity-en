G/A Custom Vars

We're setting the following custom vars at the page-scope to be associated with all events.

Slot 1 = Video Source (youtube url, or other video source url)

Slot 2 = UserId

Slot 3 = UserName

G/A Event for QUIZ

We're capturing the Answer event.

Category = "HapYak - [Project Title]"

Action = "Quiz - Correct | Incorrect"

Label = "[Question Text]"

G/A Event for LINK

We're defining LINK as anything that trigger sanother webpage to be loaded. This may be a user click, or it may be an automatic page load triggered by a Quiz answer.

Category = "HapYak - [Project Title]"

Action = "Link"

Label = "[url]"

G/A Event for NAVIGATION

We're defining NAVIGATION as anything that triggers a seek in the video (excluding a user click on the timeline). This may be a user click, or it may be an automatic seek triggered by a Quiz answer.

Category = "HapYak - [Project Title]"

Action = "Navigation"

Label = "[seek time] | Chapter: [chapter id]"
