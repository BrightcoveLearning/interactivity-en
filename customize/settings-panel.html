When editing a HapYak project, click Settings for the following options:

Video source

You can change the video that is behind the annotations.  This is used when you have a new version of the video that you'd like to use.  It is also used when you'd like to create a project that is similar to one you've already created.  You can copy the original, then simply replace the video source of the copy.

Tags

Tags are used for organizing projects into groups. You can more easily search your projects and more easily filter in reports based on tags.

Tags are also used in some direct integrations with other systems to automatically add annotations to videos like with Brightcove Plugin)

Tags should be all lower case, no spaces. (ex/ csat-question). Separate multiple tags with a comma (ex/ csat-question, onboarding). Make sure to click "Save" and then "Submit" to save the tags.

Track ID

The track is the set of annotations on a video.  This field is for reference.

Show questions

Choose to show quiz questions to the viewer only once on the first view, or every view (including repeated viewings).

Custom Annotation CSS URL (Available with Enterprise License)

Many customers apply a custom CSS to their project - for example to change the colors, fonts or other style of their annotations.  You host the CSS file (just as you would for your web pages) and then paste in the URL for that CSS here. HapYak wants your interactive videos to be beautiful and reflect your branding and style. For all Pro customers and up we provide an initial CSS file based on your guidelines. Contact success@hapyak.com.

Custom Control CSS URL (Available with Enterprise License)

This is rarely used.  When it is, it is to completely redesign the style of the HapYak player. For small changes and tweaks to the HapYak player (like removing the annotation markers), we recommend using the Custom Annotation CSS field.

String Resource URL

Used for localizing the buttons and prompts of the video.

Control Bar Style

Allows users to Hide or show Skip Buttons, Logo, Timeline, or Annotation Markers.

Thumbnail

Allows users to upload a thumbnail image for the project

Convert Project To Template

Allows users to convert any project into a template
Note: This option will permanently remove the selected project from the projects folder and turn it into a template. This action CANNOT BE UNDONE. Take note of which projects you convert or make sure to make copies of projects before you convert them.
