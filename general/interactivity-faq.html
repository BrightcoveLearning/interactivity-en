General topics

How do I upload videos?
Typically, videos are not directly uploaded to HapYak. Rather, videos remain where they always have been (an existing video host, locally hosted on a server, etc...). Users copy the link or embed code of the video and paste it into the " URL Or Embed Code" tab in the " New Project" menu on HapYak and then add annotations. If you don't have a video provider, we'd be happy to recommend one.

Alternatively, users can upload an .mp4 file directly to HapYak by choosing the " Upload" tab in the " New Project" menu and selecting the file on their system.

I don't have a video host. I only have a video file (.mp4, .mov, etc...) Can I still use HapYak?
Absolutely! Simply ensure your video lives on a server that will give you a link to the URL of the file (like Amazon S3, Dropbox, etc...). Copy the link to the file, paste into a new project in HapYak. Select any open source video player from the "embed type" list, for example videoJS.
I upload my videos directly into my LMS or CMS and don't know the video file URL. How can I access it?
When most LMSs and CMSs allow you to upload a file, they place it on a server so other users can see it. However, each one does it differently, so finding the URL might be a challenge. Please, send us an email Brightcove Support directly and we'll help you locate where the file URL is located.
